# Free Pascal GO32v2 Setup

OS: go32v2
CPU: i386

## Requirements

- **fpcupdeluxe** to handle all the bootstrapping (I use v2.4.0g, but newer versions should work too)
- CWSDPMI or another compatible DPMI (DOS Protected Mode Interface) host

## Compiler Setup

**FPC setup**

1. Open **fpcupdeluxe**
2. Find a location that is easy to find, for example:
   `E:\fpc-go32v2-i386`
3. Choose either **stable** or **3.2.2**
   I chose this version because it's proven working
4. Click install **Only FPC**
5. Wait for at least 10-15 minutes until the compile process is finished

**Cross-compiler setup**

1. Click **Cross**
2. Choose CPU: **i386**
3. Choose OS: **go32v2**
4. Then click **Install compiler**

## Current status

The GO32v2 target is experimental and tested through:

`latest\TESTS\GO32`

If you use Lazarus IDE, the project can be immediately built with `Shift+F9`
